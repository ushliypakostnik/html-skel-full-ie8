Пример страницы проекта 3

Вариант подключения: одна страница - несколько модулей

Это страница page3.html. Она создана на основе страницы app.html.
На ней происходит подключение джаваскриптового модуля Page1 (js/page1.js). Этот модуль создан на основе модуля App (js/app.js).
Также на ней происходит подключение джаваскриптового модуля Page2 (js/page2.js). Этот модуль создан на основе модуля App (js/app.js).

Для примера, пусть модуль Page1 будет делать только одну вещь: рисовать на странице красный квадрат из элемента div#testBlock1 и увеличивать его размер на 1px при каждой перерисовке (перерисовка происходит при загрузке DOM-модели, событии полной загрузки и при каждом событии изменения размера окна браузера).

Для примера, пусть модуль Page2 будет делать только одну вещь: рисовать на странице жёлтый квадрат из элемента div#testBlock2 и увеличивать его размер на 1px при каждой перерисовке (перерисовка происходит при загрузке DOM-модели, событии полной загрузки и при каждом событии изменения размера окна браузера).
