Пример страницы проекта 1

Вариант подключения: одна страница - один модуль

Это страница page1.html. Она создана на основе страницы app.html.
На ней происходит подключение джаваскриптового модуля Page1 (js/page1.js). Этот модуль создан на основе модуля App (js/app.js).

Для примера, пусть модуль Page1 будет делать только одну вещь: рисовать на странице красный квадрат из элемента div#testBlock1 и увеличивать его размер на 1px при каждой перерисовке (перерисовка происходит при загрузке DOM-модели, событии полной загрузки и при каждом событии изменения размера окна браузера).
